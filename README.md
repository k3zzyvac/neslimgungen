# Eksem Life Güzellik Merkezi

Malatya'nın kalbinde, modern estetik ve profesyonel hizmet anlayışıyla lazer epilasyon, cilt bakımı ve kalıcı makyaj hizmetleri sunan kurumsal web sitesi.

## Özellikler
- **Responsive Tasarım**: Mobil ve tablet uyumlu modern arayüz.
- **Performans Odaklı**: Next.js 15 ve App Router ile optimize edilmiş hız.
- **Premium Estetik**: Lüks ve güven veren tasarım dili.
- **İletişim Entegrasyonu**: WhatsApp ve Hızlı Arama butonları.
- **SEO Uyumluluğu**: Meta veriler ve doğru hiyerarşi ile arama motoru optimizasyonu.

## Kurulum
Bu proje [Next.js](https://nextjs.org) ile geliştirilmiştir.

```bash
npm install
npm run dev
```

## Dağıtım
Projeyi Vercel veya benzeri platformlara kolayca deploy edebilirsiniz.
